# 智能待办事项管理系统

一个功能完整的Python命令行待办事项管理工具。

## 功能特性

- ✅ 添加/删除/完成待办事项
- 🔴🟡🟢 三级优先级分类（高/中/低）
- 📅 按截止日期排序
- 🔍 关键词搜索
- 💾 JSON文件持久化存储
- 📊 统计信息查看

## 快速开始

```bash
# 运行程序
python main.py

# 按照菜单提示操作
# 1. 添加待办事项
# 2. 查看所有待办
# 3. 标记完成
# ...
```

## 使用示例

```
请选择操作 (0-10): 1

--- 添加待办事项 ---
标题: 完成项目报告
描述: 周五前提交给经理
优先级: 1.高 2.中 3.低
选择优先级 (1-3, 默认2): 1
截止日期 (YYYY-MM-DD, 默认7天后): 2026-06-01

✅ 已添加: [1] 完成项目报告
```

## 技术栈

- Python 3.8+
- 数据类（dataclasses）
- JSON持久化
- 命令行界面
